    Windows Optimisations¶

    Ultimate Windows Performance, Debloating, and Privacy Suite

    Welcome to the official documentation site for the Windows Optimisations repository. This suite provides a curated, high-performance ecosystem of registry tweaks, automated installation files (autounattend.xml), toggle scripts, and utility configurations. Whether you want to boost gaming FPS, debloat background processes, or regain complete privacy on Windows 10/11, you'll find comprehensive step-by-step guides here.

    🚀 Key Optimization Areas¶

    • Performance First

      Registry tweaks for thread priority alignment, SSD alignments, network routing, and custom power plans (QuickCPU).

    • Privacy & Telemetry

      Broad-spectrum telemetry disablement, background app execution filters, and Edge telemetry cleanups.

    • Automated OS Setup

      autounattend.xml automation templates that bypass TPM, Secure Boot, and RAM restrictions automatically.

    • Curated Toolset

      Hand-picked utilities for latency optimization (MSI Utility), custom resolutions (CRU), and hardware temperature tracking.

    🎯 Guided Paths by Use Case¶

    To help you get the most out of your optimization process, identify your use case below and follow the linked documentation paths:

    🎮 Gamers (Maximum FPS & Low Latency)¶

    1. Apply CPU priorities and the custom power plan via System Tweaks.
    2. Configure Message Signaled-Based Interrupts using the guide in Extra Utilities.
    3. Ensure display optimizations are active by consulting Display Tweaks.
    4. Update display and chipset drivers via Driver Management.

    💻 Legacy / Low-End PCs (<8GB RAM & Weak CPUs)¶

    1. Turn off visual transparency and animations using Appearance Tweaks.
    2. Configure automatic RAM cleaning with Mem Reduct Guide.
    3. Switch update routines off using Windows Update Controls if you do not use the Microsoft Store.
    4. Clean bloated caches using the Aggressive Storage Cleanup guide.

    🔒 Privacy-Minded Users (Zero Telemetry)¶

    1. Stop data collection routines using Telemetry Tweaks.
    2. Restrict systemic access permissions via Privacy Tweaks.
    3. Transition to a telemetry-free browser by reading the Web Browsers Guide.
    4. Consider performing a clean, fully-privatized OS setup using the Unattended Installation XML.

    ⚠️ Important Precautions¶

    [!WARNING] - Always perform a System Restore Point creation before running any scripts. - Some tweaks disable core security components (Windows Defender, SmartScreen). Only apply these modifications if you have alternative safeguards active or operate in secure isolated environments. - Read the Antivirus Guide to understand recommended modern scanning setups.
