# Windows and Office Activators

This directory contains tools for activating Windows and Microsoft Office products.

## Available Tools

### 1. MassGrave (MAS)
- **File**: `MassGrave.cmd`
- **Description**: Microsoft Activation Scripts (MAS) is an open-source Windows and Office activator using HWID, KMS38, and Online KMS activation methods.
- **Methods**:
  - **HWID (Hardware ID)**: Permanent activation for Windows 10/11.
  - **KMS38**: Activates Windows 10/11/Server until the year 2038.
  - **Online KMS**: Activates Windows and Office for 180 days (with renewal task).
- **Usage**: Run the `.cmd` file as administrator and follow the on-screen menu.

### 2. AAct x64
- **File**: `AAct_x64.exe`
- **Description**: A compact KMS-activator for Windows VL editions and Office.
- **Usage**: Run the executable as administrator to access the activation interface.
